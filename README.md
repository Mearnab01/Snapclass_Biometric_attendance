# SnapClass 🎓🤖

SnapClass is an AI-powered biometric attendance system built using Python, Streamlit, and PyTorch.

It uses:

- 👤 Face Recognition Attendance
- 🎤 Voice-based Attendance
- 📚 Subject Management
- 🧑‍🏫 Teacher & Student Dashboards
- 🗃️ Database Integration

## 🚀 Features

- Real-time face attendance
- Voice attendance system
- Auto student enrollment
- Attendance tracking & reports
- Modern Streamlit UI
- Modular project architecture

## 🛠️ Tech Stack

- Python
- Streamlit
- PyTorch
- OpenCV
- Speech Recognition
- SQLite

## 📦 Installation

```bash
git clone <repo-url>
cd snapclass_pytorch
pip install -r requirements.txt
```
